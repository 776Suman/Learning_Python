# Learning Python – Beginner Practice Programs

This repository contains simple and beginner‑friendly Python programs that I created during my Digital & AI Training at TPSODL.  
These examples are perfect for anyone starting their Python journey.

## 📘 What’s Included
- Even numbers program  
- Reverse string  
- Palindrome checker  
- Arithmetic operations (add, subtract, multiply, divide)  
- Conditional operators (if/else, comparisons)  
- Basic input/output examples  

## 🎯 Purpose of This Repository
To help beginners understand Python through small, easy-to-follow examples.  
Each program focuses on one concept so learners can practice step by step.
