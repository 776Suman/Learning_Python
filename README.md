# Learning Python – Beginner Practice Programs

This repository contains simple and beginner‑friendly Python programs that I created during my Digital & AI Training at TPSODL.  
These examples are perfect for anyone starting their Python journey.

## 📘 What’s Included
- Even numbers program  
- Reverse string  
- Palindrome checker  
- Arithmetic operations (add, subtract, multiply, divide)  
- Conditional operators (if/else, comparisons)  
- Basic input/output examples  

## 🎯 Purpose of This Repository
To help beginners understand Python through small, easy-to-follow examples.  
Each program focuses on one concept so learners can practice step by step.

## Module 2: Control Flow – Conditionals & Loops

This module covers the basics of decision‑making and repetition in Python.  
It includes simple, beginner‑friendly programs that explain how `if` statements and loops work.

---

### 🔹 Assignments 1–4: Conditional Statements

### 📝 Assignment 1: Simple if Statement  
Ask the user for a number and print whether it is positive.

### 📝 Assignment 2: if‑else Statement  
Ask the user for a number and print whether it is positive or negative.

### 📝 Assignment 3: if‑elif‑else Statement  
Ask the user for a number and print whether it is positive, negative, or zero.

### 📝 Assignment 4: Nested if Statement  
Ask the user for a number and print whether it is:  
- Positive and even  
- Positive and odd  
- Negative  

---

### 🔹 Assignments 5–10: Loops

### 📝 Assignment 5: for Loop  
Print numbers from 1 to 10 using a for loop.

### 📝 Assignment 6: while Loop  
Print numbers from 1 to 10 using a while loop.

### 📝 Assignment 7: Nested Loops  
Print a 5×5 grid of asterisks (*) using nested loops.

### 📝 Assignment 8: break Statement  
Keep asking the user for numbers until they enter 0.  
Then print the total sum.

### 📝 Assignment 9: continue Statement  
Print numbers from 1 to 10 except 5 using a for loop and continue.

### 📝 Assignment 10: pass Statement  
Use `pass` inside a loop as a placeholder for future logic.

---

### 🔹 Assignments 11–13: Loops + Conditionals

### 📝 Assignment 11: Even Numbers with Loop + Conditionals  
Ask the user for a number and print all even numbers from 1 to that number.

### 📝 Assignment 12: Sum of Digits  
Calculate the sum of digits of a number using a while loop.

### 📝 Assignment 13: Prime Number Check  
Check whether a number entered by the user is prime using a for loop.
